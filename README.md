# Tehc Site Engine

Tehc Site Engine is a website engine to create dynamic webpages from multiple HTML files.

## Usage

HTML:

```html
<!DOCTYPE html>
<html>
<head>
    ...
    <script src="https://cdn.jsdelivr.net/gh/Tehcn/TehcSiteEngine@main/src/engine.js">
    ...
</head>
<body>
    ...
    <main id="root">

    </main>
    ...
</body>
</html>
```

File Structure:

```txt
(root)
|   index.html
\---views
        home.html
        test.html
```
